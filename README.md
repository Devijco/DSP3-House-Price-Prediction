# House Price Prediction Project

Proyek ini berfokus pada prediksi harga rumah berdasarkan berbagai fitur menggunakan pembelajaran mesin. Proyek ini melibatkan analisis data eksplorasi (EDA), prapemrosesan data, dan melatih model regresi untuk memprediksi harga rumah secara akurat.

## Tinjauan Proyek

Tujuan utama dari proyek ini adalah untuk membangun model prediktif yang memperkirakan harga rumah. Ini termasuk memahami distribusi data, menangani nilai yang hilang, fitur penskalaan, dan memilih variabel yang relevan. Model akhir dievaluasi berdasarkan metrik kinerja untuk memastikan akurasi dalam prediksi harga.

## Analisis Utama:

- **Data Exploration:** Memahami distribusi data dan hubungan antar fitur.
- **Data Preprocessing:**
  - Menangani nilai yang hilang dan fitur penskalaan.
  - Menghapus fitur multikolinier dan memilih variabel penting.a.
- **Model Training and Evaluation:**
  - Memisahkan data menjadi set pelatihan dan pengujian.
  - Melatih model regresi berganda dan mengevaluasi kinerjanya.
    
## Dataset

*assets/train.csv and assets/test.csv* merupakan data latih dan data uji untuk pemodelan
